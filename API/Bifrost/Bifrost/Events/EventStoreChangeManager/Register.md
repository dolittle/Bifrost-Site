**Register**

Register a [Bifrost.Events.IEventStoreChangeNotifier](Bifrost.Events.IEventStoreChangeNotifier) that can be notified when changes occur in an event store

#Parameters#


##type##
Type of notifier to register, it must however implement the [Bifrost.Events.IEventStoreChangeNotifier](Bifrost.Events.IEventStoreChangeNotifier) interface
